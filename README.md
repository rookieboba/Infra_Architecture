리전: ap-northeast-2 (서울 리전)
VPC 이름: main-vpc
사용된 보안 그룹:
  - 웹 서버 보안 그룹: web-sg
  - 데이터베이스 보안 그룹: db-sg
사용된 IAM:
  - EC2 인스턴스 프로파일 및 역할: WebInstanceProfile
  - Auto Scaling을 위한 권한 설정 포함

Terraform 명령어 사용 방법

1. 환경 초기화
```
terraform init
```
Terraform 플러그인 다운로드 및 환경 초기화.

2. 계획 확인
```
terraform plan
```
인프라가 어떤 방식으로 생성될지 확인.

3. 적용
```
terraform apply
```
계획된 인프라를 실제로 생성.

4. 상태 확인
```
terraform show
```
현재 Terraform 상태 파일의 리소스 확인.

5. 정리 및 삭제
```
terraform destroy
```
생성된 인프라를 모두 제거.
